# Pattern: Min-Max Filtering

---

## 🧠 Idea
Use global min and max to filter valid elements

---

## 🔁 Technique
- Find min, max
- Count elements satisfying condition

---

## 🚨 Recognition Signals
- “strictly smaller and greater”
- “exists smaller and larger element”
- array constraint problems

---

## 🔗 Related Problems
- Remove min/max elements
- Range filtering problems
