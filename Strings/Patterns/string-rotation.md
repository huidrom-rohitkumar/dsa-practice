# Pattern: String Rotation via Concatenation

---

Idea:
All rotations of a string s are substrings of s + s

---

Technique:
- Concatenate string with itself
- Check if target exists as substring

---

Recognition Signals:
- “rotate string”
- “cyclic shift”
- “wrap-around behavior”
